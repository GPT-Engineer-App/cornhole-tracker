# cornhole-tracker

Let's make a webapp that allows me to track a current game of Cornhole, tracking both teams, each with customisable team names and colors, with a modern / creative / intuitive user interface. No logins needed; just "create a session" for the page instance.

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/cornhole-tracker.git
cd cornhole-tracker
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
